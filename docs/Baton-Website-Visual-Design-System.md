# Baton Website Visual Design System

## Goal

The website should immediately communicate that Baton is a premium music
player whose interface adapts to your music.

## Brand Strategy

Two identities coexist.

### Baton Identity

Permanent

Color: #E98345

Used for: - Logo - Primary CTA - Buttons - Navigation highlights -
Marketing

### Music Identity

Dynamic

Artwork-derived colors.

Used for: - Hero animation - Wave graphics - Demo player - Accent
glows - Hover states

## Hero

Instead of a static orange hero:

Use a slow color cycle based on sample album artwork.

Blue → Purple → Emerald → Orange → Rose

Transition every 6--10 seconds.

## Background

Use subtle radial gradients.

Dark theme:

Black → artwork accent (5--8%) → black

Light theme:

Warm white → artwork tint (3--5%) → white

## Waves

Current waves are excellent.

Enhancements:

-   Adaptive color
-   Slight parallax
-   Floating music notes
-   Playback pulse synced to animation

## Buttons

Primary - Baton Orange

Secondary - Neutral outline

Avoid making CTAs dynamic.

## Sections

Alternate subtle backgrounds:

Hero Near black

Section 2 Dark gray

Section 3 Accent tint

Section 4 Neutral

Footer Almost black

## Demo

Add a live mock player.

Every few seconds: - Album changes - App recolors - Website accents
update

This instantly demonstrates the adaptive UI.

## Motion

Slow. Elegant. Apple-like.

Avoid: - Fast gradients - Rainbow effects - Excessive blur

## Accessibility

Maintain contrast during every color transition.

## Messaging

Keep the promise clear:

"Baton is always Baton. Your music gives it personality."
